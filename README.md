## 🛠️ Installation Steps


1. open Terminal and clone the repo

```bash
  git clone https://github.com/Lakhankumawat/LearnCPP.git
```
 2. cd over 
 
```bash
   cd LearnCPP
```
 3. follow along
```
   happy coding!
```
